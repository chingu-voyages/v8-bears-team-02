# Stack Overflow Clone

## Installing Client
1. Clone Repo
2. `cd v8-bears-team-02/client`
2. install dependencies `npm install`
3. run the server `npm start`

## Installing Server
1. Clone Repo
2. `cd v8-bears-team-02/server`
2. install dependencies `npm install`
3. run the server `npm start`
